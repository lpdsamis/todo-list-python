# 📝 To-Do List Application

تطبيق بسيط لإدارة المهام باستخدام Python مع حفظ البيانات في JSON.

## ✨ الميزات
- إضافة مهام جديدة مع أولوية (عالية/متوسطة/عادية)
- عرض جميع المهام
- تصفية المهام (المكتملة/غير المكتملة)
- تحديد المهام كمكتملة
- حذف المهام
- حفظ تلقائي للمهام في ملف JSON

## 🚀 طريقة التشغيل

```bash
# تشغيل التطبيق
python todo.py
